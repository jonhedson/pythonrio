Google Code adota Mercurial para controle de versão distribuído

April 24 2009, 12:42 PM  by PythOnRio
Hoje o pessoal do Google Code anunciou a adoção do Mercurial como sistema de controle de versão distribuído.

O Mercurial é um DVCS (distributed version control system) desenvolvido em Python.

A escolha foi feita pelo Mercurial possuir menor barreira de entrada, pois conta com excelente documentação, livros online e possui ferramentas como o Tortoise Hg que simplificam o uso do sistema.

Para viabilizar este serviço, os Engenheiros do Google adaptaram o Mercurial para usar sua infraestrutura baseada em serviços HTTP e BigTable. Mais detalhes sobre a implementação poderão ser vistos em futuros TechTalks.
