JQDjangoGrid: Plugin Django para criar DataGrids com JQuery

July 24 2009, 12:39 PM  by PythOnRio
Criar DataGrids no Django com Ajax nunca foi tão fácil!

O JQDjangoGrid é um plugin duplo. Desenvolvido como uma aplicação plugável do Django, o JQDjangoGrid estende o JQuery possibilitando a criação de DataGrids Ajax.

Fácil de instalar e customizar, o plugin suporta ordenação de resultados e pesquisa. Esta primeira versão ainda não suporta os recursos de inserir e alterar registros "inline", diretamente no datagrid, mas possui um método genérico para remoção de registros e outra facilidade que retorna a Chave Primária do objeto selecionado.

Veja mais informações no post do Gustavo Henrique, desenvolvedor do plugin, que explica o caminho das pedras para utilizar o JQDJangoGrid.
