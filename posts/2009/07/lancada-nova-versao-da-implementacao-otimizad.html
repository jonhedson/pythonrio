Lançada nova versão da implementação otimizada do Python do Google

July 15 2009, 7:28 AM  by PythOnRio
O dia começou bem com a notícia da nova versão 2009Q2 do Unladen Swallow, projeto de Software Livre patrocinado e liderado pelo Google.

O projeto é uma implementação acelerada do Python. E esta versão é a primeira a utilizar LLVM para compilar funções Python para código de máquina. 

Esta versão ainda não é recomendada para uso em produção, mas os resultados já indicam que o desenvolvimento está no caminho certo. O Unladen Swallow 2009Q2 passa com sucesso em todos os testes de bibliotecas e ferramentas externas ao Python listadas em na Página de Testes, resultando em compatibilidade com projetos como Twisted, Django, NumPy e Swig.
