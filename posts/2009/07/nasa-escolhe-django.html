NASA escolhe Django

July 30 2009, 7:45 AM  by PythOnRio
Hoje recebi a notícia, através do Rafael Lima, de que a NASA escolheu o Django para desenvolver o Projeto NEBULA.

O Projeto NEBULA é um ambiente de Cloud Computing desenvolvido pelo Centro de Pesquisa Ames. O objetivo é desenvolver um ambiente de alta capacidade computacional, de armazenamento e de conectividade de rede. A estratégia utilizará virtualização focando escalabilidade para alcançar eficiência em custo e energia.

Muito estudo foi feito até a escolha do Django:

"After an extensive trade study, the NEBULA team selected Django, a
python-based web application framework, as the first and primary
application environment for the Cloud. NEBULA users have access to an
extensive collection of open-source django "apps", providing features
ranging from simple blogs, wikis, and discussion forums, to more
advanced collaboration suites, image processing, and more."

Leia mais em http://nebula.nasa.gov/services/framework

E para aprender Django visite o site Aprendendo Django.
