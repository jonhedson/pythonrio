Usando Python direto do seu Pendrive

July 11 2009, 3:59 PM  by PythOnRio

O projeto Portable Python é uma distribuição do Python totalmente portátil capaz de rodar a partir de um pendrive, dispensando o processo de instalação. Basta desempacotar os arquivos para um pendrive e usar a linguagem Python com todas as suas baterias.

A iniciativa surgiu inspirada no projeto Movable Python que tinha uma proposta semelhante, mas era pago.

A distribuição está disponível nas versões mais recentes do Python: 2.5.4, 2.6.1 e 3.0.1. E como não fosse o bastante, o pacote inclui:

  • o framework web Django 1.0.2
  • o framework wxPython para desenvolvimento de aplicações desktop
  • as bibliotecas NumPy e SciPy para computação científica
  • o framework PyGame para programação de jogos e aplicativos multi-mídia.
  • os IDEs PyScripter e SPE para facilitar a vida do programador.

Tudo isso por apenas 200 megabytes daquele seu pendrive de 16 gigabytes!

Atualmente o projeto suporta apenas a plataforma Windows, mas existe a intenção de atender outras plataformas.

Além disso, os desenvolvedores estão trabalhando para estabelecer a compatibilidade com a suite PortableApps que já conta com distribuições portáteis de aplicativos como Firefox, OpenOffice, etc.

Para aqueles que pensavam em conhecer Python, mas resistiam em instalar o interpretador e bibliotecas adicionais em seu computador: seus problemas acabaram! Literalmente!

"Happy hacking!"
